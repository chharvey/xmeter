6.7 #Atoms.font-size font-size

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Set the font-size property on an element. There are eight Atoms available, each corresponding to a font-size given by the projects type scale. You can see the source for the font-size settings in /src/__settings.less.

author
Chris Harvey
updated
2018-03-26
Example
  • Quark (0.625)
  • Atom (0.75)
  • Molecule (1.00)
  • Asteroid (1.25)
  • Planet (1.75)
  • Star (2.25)
  • Galaxy (3.00)
  • Universe (4.00)
